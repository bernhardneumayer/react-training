## How to Approach Exercises

### Difficulty Levels

- ‚≠ê **Easy**: Basic syntax, single concept
- ‚≠ê‚≠ê **Medium**: Combines 2-3 concepts
- ‚≠ê‚≠ê‚≠ê **Advanced**: Complex logic, multiple patterns
- ‚≠ê‚≠ê‚≠ê‚≠ê **Expert**: Real-world scenario, many concepts

### Recommended Order

1. **First Pass**: Do all Easy exercises to get comfortable
2. **Second Pass**: Tackle Medium exercises in order
3. **Third Pass**: Try Advanced exercises
4. **Challenge**: Complete Expert exercises

### Study Strategy

**Option 1: Sequential (Recommended for beginners)**
- Do exercises in order as they appear in files
- Each builds on previous concepts

**Option 2: Topic-focused**
- Master one topic completely before moving to next
- Good for those comfortable with basics

**Option 3: Difficulty-based**
- Do all Easy exercises first
- Then all Medium, Advanced, Expert
- Good for confidence building

## Getting Help

### When you're stuck:

1. **Read the error message** carefully (they're helpful!)
2. **Check the TODO comments** in the exercise file
3. **Review the session notes** in the main curriculum
4. **Look at similar examples** earlier in the file
5. **Check the solution** to see one approach
6. **Ask during training** or pair with another participant

### Red flags that you're stuck:

- Been on same exercise for 30+ minutes
- Error message doesn't make sense
- Not sure what the exercise is asking
- Completed it but feels wrong

Don't suffer in silence - ask for help! üôã
